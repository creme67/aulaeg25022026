## Requisitos Funcionais (RF)

### RF01 — Login do usuário
O sistema deve permitir que o usuário realize login com e-mail e senha.

### RF02 — Recuperar senha
O sistema deve permitir que o usuário solicite recuperação de senha por e-mail.

### RF03 — Atualização de perfil
O usuário pode alterar nome, foto e dados pessoais.

### RF04 — Cadastro de novos usuários
O sistema deve permitir que novos usuários se cadastrem informando nome, e-mail, senha e confirmação de senha.

### RF05 — Encerramento de sessão (Logout)
O sistema deve permitir que o usuário encerre sua sessão de forma segura a qualquer momento.

### RF06 — Gestão de Níveis de Acesso (RBAC)
O sistema deve diferenciar permissões entre usuários comuns e administradores (ex: apenas administradores podem excluir outros usuários).

### RF07 — Notificações do Sistema
O sistema deve enviar notificações ao usuário sobre eventos importantes (ex: confirmação de alteração de senha ou alertas de atividade).

### RF08 — Pesquisa e Filtragem
O sistema deve permitir que o usuário realize buscas e aplique filtros em listas de dados (ex: buscar por data, nome ou categoria).

### RF09 — Histórico de Atividades (Logs)
O sistema deve registrar as principais ações realizadas pelo usuário para fins de auditoria e segurança.

### RF10 — Exclusão de Conta (LGPD)
O sistema deve permitir que o usuário solicite a exclusão definitiva de sua conta e de seus dados pessoais, conforme as normas de privacidade.

### RF11 — Autenticação de Dois Fatores (2FA)
O sistema deve oferecer a opção de ativar uma camada extra de segurança através de um código enviado por SMS ou aplicativo de autenticação.
